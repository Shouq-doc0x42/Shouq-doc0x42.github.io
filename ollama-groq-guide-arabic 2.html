المقدمة — ما هي وكيف تستخدمها

ما هي واجهات برمجة الذكاء الاصطناعي (AI APIs)؟

تتيح لك واجهة برمجة الذكاء الاصطناعي إرسال نص إلى نموذج ذكاء اصطناعي مستضاف في السحابة، وتلقّي ردٍّ ذكي. تخيّلها كدالة برمجية تستدعيها عبر الإنترنت — ترسل سؤالاً أو تعليمة، يفكّر النموذج، ثم يعيد الإجابة.

أمثلة شائعة: OpenAI، Anthropic، Groq — كلها سحابية ومدفوعة حسب الاستخدام.

ما هي نماذج الذكاء الاصطناعي المحلية؟

تعمل النماذج المحلية على جهازك تماماً دون الحاجة للإنترنت. بياناتك تبقى عندك، أنت تتحكم في كل شيء، ولا رسوم على الاستخدام. أداة مثل Ollama تجعل هذا بسيطاً جداً باستخدام أمر واحد في سطر الأوامر.

خصوصية كاملة. يعمل بدون إنترنت. مجاني بالكامل.

لماذا Ollama و Groq تحديداً؟
Ollama

يشغّل النماذج مفتوحة المصدر محلياً. مجاني. البيانات لا تغادر جهازك. مثالي للسجلات الحساسة والتحقيقات الخاصة.

Groq API

استدلال سحابي فائق السرعة باستخدام معالجات LPU المتخصصة. يعالج مستندات ضخمة وإستفسارات فورية بزمن استجابة منخفض جداً.

كلاهما معاً

استخدم المحلي للبيانات الحساسة، وGroq حين تحتاج السرعة والحجم. ادمجهما في خطوط أمنية متكاملة.

💡
مثال من الواقع: محلل DFIR يستخدم Ollama محلياً لتحليل سجلات البرمجيات الخبيثة الحساسة (تبقى البيانات على الجهاز)، ثم يستخدم Groq لتلخيص تقرير حادثة يحوي 50,000 سطر في ثوانٍ معدودة.

التثبيت والإعداد — Ollama و Groq

🦙 الجزء الأول — إعداد Ollama

Ollama أداة مفتوحة المصدر تتيح لك تنزيل وتشغيل نماذج لغوية كبيرة على جهازك بأمر واحد. تُدير تنزيل النماذج وتشغّل خادم استدلال محلي على المنفذ localhost:11434.

النظام طريقة التثبيت
🍎 macOS حمّل المثبّت من ollama.com أو استخدم brew install ollama
🐧 Linux curl -fsSL https://ollama.com/install.sh | sh
🪟 Windows حمّل ملف .exe من ollama.com/download
TERMINAL — تشغيل أول نموذج سطر الأوامر
$ ollama run llama3
> Pulling model llama3... (التنزيل الأول ~4GB)
> Model loaded into memory
>>> Send a message ( /? for help)
$ ollama list
> NAME ID SIZE MODIFIED
> llama3:latest 365c0bd3... 4.7 GB 2 minutes ago
$ ollama pull mistral
> Pulling mistral:7b model...
ℹ
ماذا يحدث عند تشغيل ollama run llama3؟
يقوم Ollama بتنزيل أوزان نموذج LLaMA 3 (للمرة الأولى فقط)، يحمّلها في الذاكرة، يشغّل خادم استدلال محلي، ويفتح لك جلسة محادثة تفاعلية. التشغيلات التالية أسرع لأن النموذج محفوظ مسبقاً.

# 01_ollama_local_call.py
# Requires: pip install requests
# Prerequisites: Ollama installed and running (ollama serve)

import requests

# Ollama runs locally on port 11434
response = requests.post(
    "http://localhost:11434/api/generate",
    json={
        "model": "llama3.2",
        "prompt": "Explain what a reverse shell is in simple terms.",
        "stream": False
    }
)

result = response.json()
print(result["response"])
🤖 الجزء الأول-ب — تثبيت Ollama كـ Agent (خدمة دائمة)

لاستخدام Ollama من Python أو أي تطبيق خارج، تحتاج خادمه أن يعمل في الخلفية كـ agent / service دائم — بدلاً من فتحه يدوياً في كل مرة.

🐧 Linux — Systemd Service
TERMINAL — Linux Agent Setup
$ sudo systemctl enable ollama
> Created symlink /etc/systemd/system/...
$ sudo systemctl start ollama
$ sudo systemctl status ollama
> ● ollama.service - Ollama Service
> Active: active (running)
$ sudo journalctl -u ollama -f
> # View live logs
💡
تثبيت Ollama على Linux يُنشئ ملف الـ service تلقائياً. فقط شغّله مرة واحدة وسيبدأ دائماً مع إقلاع النظام.
🍎🪟 macOS / Windows — Background App
TERMINAL — macOS / Windows
# macOS: launch agent via launchd
$ ollama serve &
> Ollama server listening on :11434
> # Add to ~/.zshrc to auto-start:
$ echo "ollama serve &" >> ~/.zshrc
# Windows: runs as tray app automatically
# Or start manually:
> ollama serve
> Listening on 127.0.0.1:11434
ℹ
على macOS وWindows، تطبيق Ollama يُشغّل الخادم تلقائياً في شريط النظام (System Tray) بمجرد فتحه.
Python — التحقق من أن Ollama Agent يعمل + Chat Loop
📦
Install: pip install requests
03_ollama_agent_health_check.py
# 03_ollama_agent_health_check.py
# Simple chat with Ollama — type 'quit' to exit.
# Requires: pip install requests

import requests

MODEL = "llama3.2"

while True:
    user_input = input("You → ").strip()

    if user_input.lower() == "quit":
        print("Goodbye.")
        break

    response = requests.post(
        "http://localhost:11434/api/generate",
        json={"model": MODEL, "prompt": user_input, "stream": False}
    )

    print("\nAI →", response.json()["response"], "\n")

⚡ الجزء الثاني — إعداد Groq API

Groq خدمة ذكاء اصطناعي سحابية مبنية على معالجات LPU (وحدات معالجة اللغة) المخصصة، مما يجعلها من أسرع منصات الاستدلال الموجودة. تستخدم نفس تنسيق API الخاص بـ OpenAI، لذا التكامل سهل جداً.

  • 1
    إنشاء حساب مجاني

    زر console.groq.com وسجّل بإيميلك. الخطة المجانية كافية جداً للتعلم والمشاريع الصغيرة.

  • 2
    توليد مفتاح API

    اذهب إلى API Keys في لوحة التحكم ← اضغط Create API Key ← أعطه اسماً ← انسخ المفتاح فوراً (يظهر مرة واحدة فقط).

  • 3
    تخزين المفتاح بأمان

    لا تضعه أبداً في الكود مباشرة. استخدم متغير بيئي أو ملف .env. راجع النمط الآمن أدناه.

  • 4
    تثبيت حزمة Python

    نفّذ: pip install groq — تتكفّل حزمة Groq الرسمية بالمصادقة وإعادة المحاولة والبث.

⚠
تحذير أمني مهم: لا تنشر مفاتيح API أبداً على GitHub. أضف .env إلى ملف .gitignore. تعامل مع مفاتيح API كأنها كلمات مرور — إذا سُرّبت، أعد توليدها فوراً.
Python — إعداد Groq API بشكل آمن
📦
Install: pip install groq python-dotenv
02_groq_api_setup.py
# 02_groq_api_setup.py
# Requires: pip install groq python-dotenv
# Setup: Create a .env file with GROQ_API_KEY=gsk_your_key_here

import os
from groq import Groq
from dotenv import load_dotenv

load_dotenv()  # Load variables from .env file

client = Groq(
    api_key=os.getenv("GROQ_API_KEY")
)

# Simple API call
response = client.chat.completions.create(
    model="llama-3.1-8b-instant",
    messages=[
        {
            "role": "user",
            "content": "What are the causes of malwares in workplace?"
        }
    ]
)

print(response.choices[0].message.content)
💡
النماذج المتاحة على Groq: llama3-8b-8192، llama3-70b-8192، mixtral-8x7b-32768، gemma2-9b-it. النموذج 70b أذكى؛ النموذج 8b أسرع.

الذكاء الاصطناعي المحلي والخصوصية

🛠️

بناء أدوات صغيرة

اصنع مساعدات سطر أوامر، محلّلات سجلات، مولّدات تقارير، أو أدوات فرز التنبيهات — كلها مدعومة بالذكاء الاصطناعي بكود بسيط.

سيناريو عملي — دراسة حالة DFIR

حادثة أمنية نشاط مشبوه مكتشف — جهاز داخلي

يتلقى محلل أمني تنبيهاً عند الساعة 2:47 صباحاً. جهاز كمبيوتر داخلي يُولّد حركة مرور صادرة غير معتادة. لديه بيانات syslog خام، وقائمة عمليات، ووصف مختصر لسلوك برمجية خبيثة. يحتاج إلى الفرز السريع وإعداد تقييم أوّلي.

Python — محلّل السجلات باستخدام Ollama (محلي / خاص)
📦
Install: pip install requests
04_log_analyzer_ollama.py
# 04_log_analyzer_ollama.py
# Requires: pip install requests
# Prerequisites: Ollama running — ollama pull llama3.2

import requests

# Suspicious log snippet (stays on the local machine)
suspicious_log = """
[2025-03-29 02:47:13] WARN: Process 'svchost.exe' spawned by 'explorer.exe'
[2025-03-29 02:47:14] NET: Outbound connection to 185.220.101.47:4444 (TOR exit node)
[2025-03-29 02:47:15] FS: File created at C:\\Temp\\payload32.exe
[2025-03-29 02:47:16] REG: HKLM\\Software\\Microsoft\\Run -> payload32.exe added
[2025-03-29 02:47:18] PROC: cmd.exe /c whoami && net user && ipconfig /all
"""

system_prompt = "You are a DFIR analyst assistant. Analyze security logs, identify Indicators of Compromise (IOCs), MITRE ATT&CK techniques, and recommended actions."

user_message = f"Analyze these suspicious logs and provide: 1) Summary of what happened, 2) MITRE ATT&CK techniques, 3) Recommended immediate actions:\n\n{suspicious_log}"

response = requests.post(
    "http://localhost:11434/api/generate",
    json={
        "model": "llama3.2",
        "prompt": f"[SYSTEM]: {system_prompt}\n\n[USER]: {user_message}",
        "stream": False
    }
)

analysis = response.json()["response"]
print("=== AI Analysis ===")
print(analysis)
Python — مولّد تقارير الحوادث باستخدام Groq API (سريع / سياق كبير)
📦
Install: pip install groq python-dotenv
05_incident_report_groq.py
# 05_incident_report_groq.py
# Requires: pip install groq python-dotenv
# Setup: Create a .env file with GROQ_API_KEY=gsk_your_key_here

from groq import Groq
import os
from dotenv import load_dotenv

load_dotenv()
client = Groq(api_key=os.getenv("GROQ_API_KEY"))

malware_behavior = """
Executable dropped in C:\\Temp. Established persistence via Run registry key.
Connected to a known TOR exit node on port 4444. Performed system reconnaissance
(whoami, net user, ipconfig). Possible C2 channel established.
"""

response = client.chat.completions.create(
    model="llama-3.3-70b-versatile",
    messages=[
        {
            "role": "system",
            "content": "You are an incident response analyst. Generate structured and professional incident reports in English."
        },
        {
            "role": "user",
            "content": f"Generate a formal incident report for this malicious behavior:\n{malware_behavior}"
        }
    ]
)

report = response.choices[0].message.content
print(report)

# 06_universal_ai_query.py
# Requires: pip install requests groq python-dotenv

import os
import requests
from groq import Groq
from dotenv import load_dotenv

load_dotenv()

def query_ai(prompt, use_local=True, model="llama3.2"):
    """
    Universal function: send a query to Ollama (local) or Groq (cloud).
    use_local=True  -> Ollama (private, free, works offline)
    use_local=False -> Groq  (fast, cloud-based, larger context)
    """
    if use_local:
        # Local Ollama
        response = requests.post(
            "http://localhost:11434/api/generate",
            json={"model": model, "prompt": prompt, "stream": False}
        )
        return response.json()["response"]
    else:
        # Cloud Groq
        client = Groq(api_key=os.getenv("GROQ_API_KEY"))
        resp = client.chat.completions.create(
            model="llama-3.3-70b-versatile",
            messages=[{"role": "user", "content": prompt}]
        )
        return resp.choices[0].message.content

# Usage examples:
result = query_ai("Analyze suspicious activity",  use_local=True)
result = query_ai("Generate an incident report",   use_local=False)

# ai_router.py  — save with this exact filename
# Requires: pip install requests groq python-dotenv

import os
import requests
from groq import Groq
from dotenv import load_dotenv

load_dotenv()

class AIRouter:
    """
    Hybrid AI router — directs queries to Ollama (local) or Groq (cloud)
    based on user preference. Both backends share the same interface.
    """

    BACKENDS = {
        "local":  {"label": "Ollama (Local)",  "color": "🟢"},
        "cloud":  {"label": "Groq (Cloud)",   "color": "🔵"},
    }

    def __init__(self, backend: str = "local"):
        if backend not in self.BACKENDS:
            raise ValueError(f"backend must be 'local' or 'cloud', got: {backend}")
        self.backend = backend
        self.groq_client = None
        if backend == "cloud":
            self.groq_client = Groq(api_key=os.getenv("GROQ_API_KEY"))
        info = self.BACKENDS[backend]
        print(f"{info['color']} AI Router initialized — {info['label']}")

    def query(self, prompt: str, system: str = None) -> str:
        """Send a query to whichever backend is active."""
        if self.backend == "local":
            return self._query_ollama(prompt, system)
        else:
            return self._query_groq(prompt, system)

    def _query_ollama(self, prompt: str, system: str = None) -> str:
        # Combine system + user prompt for Ollama
        full_prompt = f"[SYSTEM]: {system}\n\n[USER]: {prompt}" if system else prompt
        r = requests.post(
            "http://localhost:11434/api/generate",
            json={"model": "llama3.2", "prompt": full_prompt, "stream": False},
            timeout=120
        )
        r.raise_for_status()
        return r.json()["response"]

    def _query_groq(self, prompt: str, system: str = None) -> str:
        messages = []
        if system:
            messages.append({"role": "system", "content": system})
        messages.append({"role": "user", "content": prompt})
        resp = self.groq_client.chat.completions.create(
            model="llama-3.3-70b-versatile",
            messages=messages
        )
        return resp.choices[0].message.content

    def switch(self, new_backend: str):
        """Hot-swap backend without restarting the app."""
        self.__init__(new_backend)

    def status(self) -> dict:
        """Return current backend info."""
        return {"backend": self.backend, **self.BACKENDS[self.backend]}

from ai_router import AIRouter


def choose_backend() -> str:
    print("\n╔══════════════════════════════════╗")
    print("║   🛡️  DFIR Security Assistant    ║")
    print("╚══════════════════════════════════╝\n")
    print("Select AI backend:")
    print("  [1] 🟢 Local  — Ollama (private, offline, free)")
    print("  [2] 🔵 Cloud  — Groq   (fast, larger context)\n")

    while True:
        choice = input("Enter 1 or 2: ").strip()
        if choice == "1": return "local"
        if choice == "2": return "cloud"
        print("  ⚠  Please enter 1 or 2.")


def main():
    backend = choose_backend()
    ai = AIRouter(backend=backend)

    SYSTEM = "You are a DFIR security analyst. Analyze threats, extract IOCs, and map to MITRE ATT&CK. Be concise and structured."

    print(f"\n✅ Ready! Using {ai.status()['label']}")
    print("   Type 'switch' to change backend, 'quit' to exit.\n")

    while True:
        query = input("🔍 Query → ").strip()
        if not query: continue

        if query.lower() == "quit":
            print("Goodbye."); break

        if query.lower() == "switch":
            new = "cloud" if ai.backend == "local" else "local"
            ai.switch(new)
            continue

        try:
            print("\n⏳ Analyzing...\n")
            result = ai.query(query, system=SYSTEM)
            print(f"{'─'*50}\n{result}\n{'─'*50}\n")
        except Exception as e:
            print(f"❌ Error: {e}\n")


if __name__ == "__main__":
    main()

from flask import Flask, request, jsonify, render_template_string
from ai_router import AIRouter

app    = Flask(__name__)
router = AIRouter(backend="local")  # default: local

SYSTEM = "You are a DFIR analyst. Identify threats, IOCs, and MITRE techniques. Be concise."

HTML = """
<!DOCTYPE html><html><body style="font-family:monospace;padding:2rem;background:#0d0f14;color:#e8eaf0">
<h2>🛡️ DFIR Hybrid Assistant</h2>
<p>Backend: <strong id="b">Local (Ollama)</strong>
  <button onclick="toggleBackend()" style="margin-left:12px;padding:4px 12px;cursor:pointer">Switch</button></p>
<textarea id="q" rows="4"
  style="width:100%;background:#1a1e28;color:#e8eaf0;border:1px solid #333;padding:8px;font-family:monospace"
  placeholder="Enter security query..."></textarea>
<br><br>
<button onclick="ask()" style="padding:8px 20px;background:#00d084;color:#000;border:none;cursor:pointer;font-weight:bold">
  Analyze</button>
<pre id="r" style="background:#0a0c11;padding:1rem;margin-top:1rem;min-height:100px;white-space:pre-wrap;border:1px solid #222"></pre>
<script>
  let backend = 'local';
  async function ask() {
    document.getElementById('r').textContent = '⏳ Analyzing...';
    const res = await fetch('/query', {
      method: 'POST', headers: {'Content-Type': 'application/json'},
      body: JSON.stringify({prompt: document.getElementById('q').value, backend})});
    const data = await res.json();
    document.getElementById('r').textContent = data.result || data.error;
  }
  async function toggleBackend() {
    backend = backend === 'local' ? 'cloud' : 'local';
    document.getElementById('b').textContent = backend === 'local' ? 'Local (Ollama)' : 'Cloud (Groq)';
    await fetch('/switch', {method: 'POST', headers: {'Content-Type': 'application/json'},
      body: JSON.stringify({backend})});
    document.getElementById('r').textContent = 'Switched to: ' + backend;
  }
</script></body></html>
"""

@app.route("/")
def index(): return render_template_string(HTML)

@app.route("/query", methods=["POST"])
def query():
    data = request.get_json()
    try:
        result = router.query(data["prompt"], system=SYSTEM)
        return jsonify({"result": result, "backend": router.backend})
    except Exception as e:
        return jsonify({"error": str(e)}), 500

@app.route("/switch", methods=["POST"])
def switch():
    new_backend = request.get_json()["backend"]
    router.switch(new_backend)
    return jsonify({"switched_to": new_backend})

if __name__ == "__main__":
    app.run(debug=True, port=5000)
